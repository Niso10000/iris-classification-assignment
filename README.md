# Iris Classification (ביו-לינגואלי)

פרויקט סיווג פרח Iris לסטודנטים בקורס Advanced AI.

Project for Advanced AI course: classify Iris flowers using KNN, SVM, Random Forest and Logistic Regression.

## התקנה / Installation

```bash
pip install -r requirements.txt
```

## הפעלה / Run

```bash
python src/classifier.py
```

## מבנה פרויקט / Project Structure

- `data/iris.csv` - הנתונים הראשוניים
- `src/classifier.py` - קוד ראשי להרצה
- `src/evaluate.py` - חישוב מדדים ומטריצות
- `src/visualizations.py` - יצירת גרפים
- `outputs/` - קבצי תמונות ו־CSV שנוצרים
- `report/report.md` - דוח אקדמי
- `PRD.md` - דרישות מוצר
